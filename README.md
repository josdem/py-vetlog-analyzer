Python Vetlog Analyzer
----------------------------

Data analysis for [Vetlog](https://vetlog.org/) database

#### Requirements

- Python version `3.11.9` or above

#### To format code

```bash
python -m black "python" "tests"
```

#### To run tests

```bash
python -m unittest discover -s tests
```

or

```bash
python3 -m unittest discover -s tests
```

## Setup

### Prerequisites

- Python 3.12+
- uv

### Installation

1. Clone the repository

```sh
git clone hhttps://github.com/josdem/py-vetlog-buddy
cd py-vetlog-buddy
```

2. Install dependencies

```sh
uv sync
```

## Usage

```sh
uv run filter
uv run suspicious
uv run vaccines
```
